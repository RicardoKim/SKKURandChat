# SKKURandChat

## 성균관대학교 학생들을 대상으로 한 랜덤 채팅 웹입니다.

## 프로그램 요구사항
- Java : 11
- Node : 8.4.1

## 프로그램 실행

### 프로그램 실행 전 확인사항
- yarn 설치

    https://heropy.blog/2017/11/25/yarn/
- concurrently 설치

    ```shell
    npm install -g concurrently
    ```
### 프로그램 실행 방법
```shell
cd https://github.com/RicardoKim/SKKURandChat.git
npm run dev
```
